Theory

MATH time...

PID tracking, we use an algorithm to detect, predict and change to a proportionate direction of path for the robot to go. This allows for the robot to run faster due to this predictive system which "predicts" the future. Weight theory, we add a gyroscope system to accurately sense, detect, change the internal momentum of the robot to turn, stop and run soomthly. Gear changer, we learn how to use code to predict the path with PID which then changes the power and speed of the rovbot using a gear changer and greatly improve the efficiency of the robot.
